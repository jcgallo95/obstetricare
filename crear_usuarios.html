ObstetriCare

Herramienta de configuración inicial — Crear usuarios de prueba

1
Crear obstetra (tu usuario)

Crea tu cuenta como obstetra. Este es el usuario con el que vas a ingresar al panel profesional.

2
Crear médico de guardia

El médico de guardia se crea directamente en la base de datos. Copiá el SQL y ejecutalo en Railway → Postgres → Database → Data.

3
Crear paciente de prueba

Primero tenés que completar el Paso 1 para obtener el ID del obstetra. Luego hacé clic acá.

4
Probar login

Verificá que los usuarios se crearon correctamente.
